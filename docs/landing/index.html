Why This Exists
Zalo automation is hard. We fixed that.
75M users, zero official bot API for personal accounts. Until now.

✕ Without OpenClaw

  • ✕ No official bot API for personal Zalo
  • ✕ Reverse-engineering protocol changes
  • ✕ Manual cookie management & session handling
  • ✕ No access control — anyone can trigger actions
  • ✕ Building from scratch every time

✓ With OpenClaw Zalo Plugin

  • ✓ QR scan login, auto-persist credentials
  • ✓ 147 pre-built actions — messaging, groups, polls, reminders
  • ✓ Built-in access control: allow/deny lists, pairing mode
  • ✓ Mention gating, output filtering, thread sandbox
  • ✓ Event-driven, plugin architecture, zero build step
Features
Everything you need to automate Zalo
147 actions across 15 categories. Fully typed. Auto-resolves names to IDs.
💬

Rich Messaging

Text, images, files, videos, voice, stickers, cards. Bold, italic, underline, colors. Urgency levels.

👥

Group Management

Create groups, manage members, admins, settings, invite links, pending approvals. 22 group actions.

🔒

Access Control

DM policies (open/pairing/allowlist/disabled). Per-group allow/deny lists. Mention gating.

📊

Polls & Reminders

Create polls with multi-choice, anonymous voting, expiry. Set reminders with repeat schedules.

🤖

AI Agent Integration

Direct integration with OpenClaw agent framework. Context assembly, quote extraction, image analysis.

⚡

Zero Build Step

TypeScript loaded directly by OpenClaw runtime. No compilation. Edit, restart, done.

💼

Business Tools

Product catalogs, quick messages, auto-replies, contact labels, notes. Full commerce support.

🛡️

Safety Built-In

Output filtering, thread sandboxing, sensitive content redaction. Block users at Zalo or bot level.

↩️

Reply Context

When users reply to messages, AI receives the quoted text + sender. Full conversation context.

How It Works
Four steps to Zalo automation
From zero to running bot in under 5 minutes.
1

Install Plugin

Clone repo, npm install, register with OpenClaw

2

Scan QR

Login with your Zalo app. Credentials auto-saved.

3

Configure

Set DM policy, group rules, mention gating in JSON config.

4

Automate

AI agent handles messages, executes 147 actions on your behalf.

# Install
git clone https://github.com/monasprox/opclaw-zalo.git
cd opclaw-zalo && npm install

# Register & start
openclaw plugins install --link .
openclaw gateway restart

# Login via QR
openclaw channels login --channel opclaw-zalo
